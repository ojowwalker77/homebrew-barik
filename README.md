# Homebrew Tap for Barik

This is the official [Homebrew](https://brew.sh) tap for [Barik](https://github.com/ojowwalker77/barik), a lightweight macOS menu bar replacement.

## Installation

```bash
brew tap ojowwalker77/barik
brew install --cask barik
```

## Updating

```bash
brew upgrade --cask barik
```

## Uninstall

```bash
brew uninstall --cask barik
brew untap ojowwalker77/barik
```
